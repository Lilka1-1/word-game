# 🎮 Its ME — игра про фразы и угадывание авторства

https://github.com/Lilka1-1/Its-ME

## 📌 Описание

Its ME — это онлайн многопользовательская игра, в которой игроки пишут какие-то фразы, а остальные должны угадать, чьи это фразы.

Дальше начинается основная часть игры:

- Фразы игроков показываются по одной в случайном порядке  
- Все игроки (кроме автора фразы) должны угадать, кому принадлежит фраза  
- Для голосования отображаются никнеймы игроков  
- На голосование даётся ограниченное время (~1 минута)  
- После завершения голосования раскрывается правильный ответ  
- Очки начисляются игрокам, которые угадали правильно  
- Раунды повторяются, пока не закончатся все фразы  
- В конце показывается таблица результатов  

## 🧠 Основная логика игры

1. Фаза ввода (30 секунд)
   - Игроки пишут свои фразы

2. Фаза показа фраз
   - Фразы перемешиваются
   - Каждая фраза показывается отдельно

3. Фаза голосования
   - Игроки выбирают, кто автор фразы
   - Автор не голосует

4. Подсчёт очков
   - За правильный ответ начисляются баллы

5. Финальный экран
   - Итоговая таблица результатов

## 🛠 Технологии

- Backend: FastAPI
- WebSocket: real-time взаимодействие
- Frontend: HTML / CSS / JavaScript
- Database: SQLAlchemy
- Async: asyncio
